# sqrt

## sqrt(var x): var 

Square root of x.

### Parameters:

**x** \- any **var**.  

### Returns:

Square root of x. 

### Example:

```c
x = sqrt(2); // x is now 1.414
x = sqrt(64); // x is now 8 
x = sqrt(1); // x is now 1
```
